A basic KAanRAit syllable block is made up of three parts, read from top to bottom: the leading consonant (top third), the vowel (middle third), and the trailing consonant (bottom third). The vowel section is split further into two halves. The half extending to the top left is used when there is a leading consonant, and the half going to the bottom right is used when there is a trailing consonant. If neither exist, both are used. If both exist, they are typically the same unless a diphthong is used, in which case the vowel extending to the top left is read first, followed by the one going to the bottom right. The common diphthongs can be found above. The phonetics section above that shows the characters that represent each sound, as well as the keys to use to type them. All the characters in the font are zero width so each syllable will start with a space. For best effect (not splitting words on line-breaks), use a non-breaking space. In English, words will not uncommonly have a consonent blend at the beginning or end of a syllable. This is resolved in KAanRAit by using consonant blend combiners, characters that connect another block without a vowel to the beginning of the main block, such as in  b' L}]n "D (blend), a single syllable made up of three blocks, the first, the leading block, with the phoneme "b", connected to the main block "LΕεn" (typed "L}]n"), connected to the trailing block "D". In a leading block, a single consonant is placed on the bottom, and in a trailing block, it is placed on the top. If two consonants are required in a consonant blend block, such as in  St' R}e/ "<s (strengths), the first one is placed on top, and the last one is placed on bottom, no matter if it is leading or trailing.
